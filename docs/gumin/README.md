该世界观的角色大多数为偶像或者coser等与ACG亚文化相对密切的职业。

早期的人物故事比较独立，各自形成独立的小系列。部分角色根据卡面和副称号代表不同时期的故事。

从Amazon开始，GUMIN所属的角色开始形成系列故事“魔大陆篇”。在继承了部分GUMIN旧角色的设定和背景的基础上继续连载。目前魔大陆篇第一阶段的故事已经于Chunithm Crystal Plus完结。

第二阶段的故事为“关西篇”，舞台为魔大陆篇的世界观底下的大阪。
