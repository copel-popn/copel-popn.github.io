# Linked VERSE

此模式为Chunithm于X-VERSE版本引入的游玩模式。

通过完成游戏内隐藏的条件，就会出现“大门”，挑战成功将会解锁隐藏曲。

每个大门各自对应了以前的版本，对应的曲目和人物也对应了当时版本的BOSS曲。

在此模式中，将由新角色“<font color="#FF00FF">尤尼</font>”作为玩家的向导。

此模式的玩法和内容设置与maimai FiNALE的PANDORA BOXXX十分相似。

官方于X-VERSEX更新前夕，于官网上公布了直到XVERSE版本结束为止所有的剧情内容。

后续的更新对话也将于此刊载。



## 背景介绍

这个世界存在着无数的次元。其中一个，存在着跟大众认知完全不同的“CHUNITHM”。

不同于各位玩家所玩到的游戏机，

那是个连接着不同<ruby>次元<rp>（</rp><rt>VERSE</rt><rp>）</rp></ruby>，从这些世界的故事中产生乐曲并传递出去的娱乐系统。

为了区别<font color="#FF00FF">尤尼</font>所描述的CHUNITHM，<font color="#FF00FF">尤尼</font>所在世界的CHUNITHM将会称为<font color="#00c777">UHUNITHM</font>。


<br>

然而，从某个时间开始，<font color="#00c777">UHUNITHM</font>却再也连接不到其他的<ruby>次元<rp>（</rp><rt>VERSE</rt><rp>）</rp></ruby>了，所以，也就再也没有新的乐曲诞生。

身为<font color="#00c777">UHUNITHM</font>的向导AI，<font color="#FF00FF">尤尼</font>为了寻求帮助，开始寻找起存在着CHUNITHM的其他<ruby>次元<rp>（</rp><rt>VERSE</rt><rp>）</rp></ruby>。

最终，她发现了这个<ruby>次元<rp>（</rp><rt>VERSE</rt><rp>）</rp></ruby>。

<br>

<font color="#FF00FF">尤尼</font>虽然尝试与之连接，然而两个世界相异的CHUNITHM却由于互相干涉，产生了系统错误。

乐曲也产生了错误，出现了“<font color="#a052ff">旋律异化</font>现象”。

只靠<font color="#FF00FF">尤尼</font>一个人是无法修复大门的，她也就这样被困在了CHUNITHM框体之中。


<br>

为了寻找回去的方法，

<font color="#FF00FF">尤尼</font>向偶然路过游玩CHUNITHM的玩家——也就是你，发起了求助。

## 关系图与用语说明




| 用语   | 说明 |
| ----------- | ----------- |
VERSE|代表无穷无尽的不同“<ruby>次元<rp>（</rp><rt>VERSE</rt><rp>）</rp></ruby>”。每个<ruby>次元<rp>（</rp><rt>VERSE</rt><rp>）</rp></ruby>都是独立的存在，<font color="#FF00FF">尤尼</font>原本所在的次元，玩家所在的次元，也是其中的一部分。
<font color="#00c777">UHUNITHM</font>|存在于<font color="#FF00FF">尤尼</font>的<ruby>次元<rp>（</rp><rt>VERSE</rt><rp>）</rp></ruby>中的娱乐系统。能够连接各个不同的<ruby>次元<rp>（</rp><rt>VERSE</rt><rp>）</rp></ruby>，提取其中的故事与情感从而生出乐曲。<br>虽然也称之为CHUNITHM，但是与玩家所熟知的CHUNITHM实在是差距过大，所以才如此称呼以作区分。
<font color="#a052ff">オルタリズム(旋律异化)</font>|代指因为BUG的缘故产生了变化的乐曲，也可以指这种现象。<br><font color="#FF00FF">尤尼</font>在与玩家所在的<ruby>次元<rp>（</rp><rt>VERSE</rt><rp>）</rp></ruby>连接的时候产生了BUG，也让游戏中的乐曲发生了这样的现象。
大门/Linked GATE|用于连接不同的<ruby>次元<rp>（</rp><rt>VERSE</rt><rp>）</rp></ruby>的大门。<br>如果是稳定状态的大门的话，<font color="#FF00FF">尤尼</font>可以从中提取出传送的能量。<br>由于<font color="#FF00FF">尤尼</font>在与玩家的<ruby>次元<rp>（</rp><rt>VERSE</rt><rp>）</rp></ruby>连接的时候出现了系统错误，全部的门都被破坏了。
修复大门|修复连接不稳定的大门的工作。<br>当玩家游玩对应的游戏时，<font color="#00b8ee">NAVI</font>与<font color="#FF00FF">尤尼</font>便会在后方对乐曲进行分析，最终获得代码。<br>只要玩家能够完成整首乐曲，便可以获得代码，修复大门。<br>当大门全部都被修复，能量也恢复供应，<font color="#FF00FF">尤尼</font>就可以回到原本的世界了。
代码|用于修复“大门”所需要的，类似密码一样的东西。<br>通过<font color="#FF00FF">尤尼</font>与<font color="#00b8ee">NAVI</font>的分析便可获得。
升级补丁|通过解析<font color="#a052ff">旋律异化</font>现象获得的代码从而生成的产物。只有安装升级补丁，才可探索更多的“大门”。
