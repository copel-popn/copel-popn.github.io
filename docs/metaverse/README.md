## 名词解释

部分用语来自于官方OST《Observe the Metaverse》。

### 电脑世界篇相关

Metaverse:是由2015年由人类所提倡的，运用量子运算技术配合环境预测模拟器组合而成的假象空间。<BR>完成之后，该空间便基于次世代网络的基础设施建设投放向整个世界。<br>2020年，由复合企业群设立的研究机关“工厂”所管理。<BR>从此以后，为了提高模拟环境的精度，开始对当时的旧人类进行将人体量子数据化的实验。<BR>直到2050年的时候，Metaverse已经成长到了不需要人类的干涉便可自行完成修复/扩建的机械生命体的等级，并被赋予了从根源上观测地球圈的人类的社会活动的职责。

框架主脑（メインフレーム）：管理运行Metaverse技能的中央电脑构造体。<br>具有不依靠人类，靠自身就能进行思考和判断的，拥有高度知性的，从某种角度可以誉为神的存在。<BR>根据记录，可以得知于2050年所发生的被称作“大灾厄”的战争正是框架主脑的决定产生。<BR>储存着复数个代理人格，而作为初期代理人格被选出来的系统名称被叫做“艾克雷尔”。<BR>大灾厄以后，便转向了保存仅存的人类以及整备地上环境的两大目的。<br>在其数据中保存了无数的世界线的模拟记录，这难道也是对人类的进化路线的预测吗？

“工厂”：：负责对Metaverse进行开发作业和管理运营的研究机关。在大灾厄之后便成为了框架主脑旗下的中间管理机构。内部的职员很大一部分是曾经身为人类的研究员量子化后的产物。（从之后的剧情可以得知，职员全部都属于被称作KHD的企业联合体之中）。<BR>由于演算的量无止境地扩大导致框架主脑的运算也陷入了迟钝状态，工厂失去了和框架主脑的连接。从此以后便在Metaverse内独立行动。

“最古”：自Metaverse生成，艾克雷尔创世完成后离开，选举形成的12个程序体。对外统称Metaverse第二世代管理系统“ORIGIN”。具有最高的权限可以进行管理。各自具有不同的职能和能力。根据其设计和能力有着各自的称号。

>已人物化的有：统制主·提丰、探求者·远古之蓝、调律者·WISEMAN、哲学主·捷夫提、裁定者·塞拉菲塔、再生者·迪安、全天之支配者·雪儿<br>仍未人物化：斗争主·盖提亚、大地支配者·吉斯库克、确率主·西克扎尔<br>特殊情况：净化者·布利斯利希德、征服者·霍尔伯格

提亚马特侵攻：于西历2065年以后发生于Metaverse系统的，来自于外部的攻击。<br>虽然在统制主·提丰的猛攻下终于将其击退，但是也让框架主脑的部分机能发生了问题。<BR>从此以后，Metaverse便加速着向混沌的方向发展。

化身人类（アバター人類）：指那些适应了Metaverse内部环境的人类。也是后续新人类的祖先。<BR>分成了数个种类，在Metaverse中除了正常的繁衍后代的方式进行的进化，**还有由框架主脑或者“工厂”所进行的，经历了人工的交配和数据的混杂等等实验才制造出来的产物**，这些产物和新人类的祖先之间的相关性仍有待考证。

“涅墨西斯”：由“大灾厄”时期人类方的兵器碎片分裂而成的恶性程序，其中诞生了一共七体的，具有自我意识的高级个体，他们自称为“混沌之器”。本身具有感染别的程序的能力。目的只有将Metaverse毁灭。

VOX：Metaverse中用来管理虚拟世界的功能单元。整体为箱子形状。用途众多，从领域的创造，建设，统治，管理，甚至到个体内部的数据改写都可以做到。正常情况下是根据操作的程序提供相应的权限，无法逾越。

精神复制体（アストラルコピー）：由Reconnect the Metaverse篇引入的概念名词。可以简单理解为个体的精神和记忆数据的整体在Metaverse这个电脑世界里的叫法，也就是“灵魂”。只要这个东西还完好存在的话，就可以经过一些手段重新复活，与之相对的，对精神复制体造成损伤的话会导致记忆或者人格丢失等不可逆的伤害。

艾崔兰克（エテメンアンキ）：框架主脑方的大型防卫据点，在Reconnect the Metaverse篇出现。是守护Metaverse最根基领域“佛诺·泽尼斯”的最终防线。原名取自于新巴比伦王朝时期建设起来的圣塔Etemenanki，又名七曜塔。

佛诺·泽尼斯（フォノ・ゼニス）：对应单词phono和zenith。phono为拉丁语的根词，代表声音的意思。zenith则为天顶，最高点之意。该领域是组成Metaverse整个世界的根基。储存着量子化的人类数据，以及提供他们生存的虚拟世界的数据，同时也是人类回归地上计划的关键。一旦被攻陷或者被破坏，将导致整个Metaverse的数据化为乌有。

### 地上篇相关

“大灾厄”之后的地球：人类在被Metaverse的管理者“框架主脑”制造的机械兵器驱逐之后，Metaverse开展了创造合成人类重建地上环境的计划。<br>当地球上的真人们在“最古”的管理下修复自然的时候，在Metaverse内部发生了内乱和恶性病毒攻击，在“最古”之一的“统制主·提丰”倒台后，Metaverse失去了对地上真人的管控能力，于是决定放权，并将真人的管理交给了手下的“机械种”，并赋予了真人一定程度自我意识保证他们能继续执行恢复地上环境的任务。

真人：由“大灾厄”被驱逐掉人类之后由机械所诞生的合成人类，前身是战前由企业KHD以不人道实验所开发的机器人经过迭代发展而来。目前担负着为人类再造地球环境的职责，但是由于电脑世界的崩坏，出现了一定的异常。<br>为Metaverse方的管理者“框架主脑”在驱逐人类之后制造出来的合成人类，由人工子宫进行生产，外观和职能在被制造出来的时候就已经固定，不会衰老。<br>真人在对机械的亲和力，身体机能以及适应性方面要胜于旧人类。<br>然而为了保证其不会产生反叛的风险，在设计之初就故意缩短了其寿命，并且剥夺了生殖能力。由于真人常常要接触被污染而十分恶劣的大地环境，所以在可存活年限本就很短的情况下，许多的真人甚至都活不到寿终正寝的时候。不但如此，当地上的再生进度达到几乎完成的时候，将会关闭人工子宫的机能，停止生产，同时自动触发真人体内的自杀因子，直接让其失去机能。可以说是从一开始就将真人当做了即用即弃的工具。

真人的历史：曾经的真人由“框架主脑”的最古进行直接掌控，通过欧米茄·昆提斯以及后继机“马尔库特的女神”这样的机械负责监督。最初的管理者是“大地支配者·吉斯库克”。在提亚马特侵攻之后提丰接替了他的工作。<BR>当提丰的统治倒台之后，剩余的最古由于乏于支出资源和精力管理地上的再生工作，于是将管理权下放到了“机械种”身上，并且赋予了真人一定量的自我意识以保证其正常执行任务。<BR>在漫长的历史发展之中，真人也渐渐地觉醒了自我，在漫长的时间之中与机械种进行着看不见的斗争行动。最终，当人类终于从Metaverse之中归来的时候，一部分强硬派真人自称为“革新者”，举起了反旗，发起了对造物主，以及本该服侍的，造就了真人这一切因缘历史的人类的反抗。

“归还种”：真人对从Metaverse中归来的人类的称呼。<BR>人类在Metaverse之中经历分分合合，无穷无尽的战争和冲突之后，最终获得了和平，同时也获得了“复苏之门”的认可，得以开启前往地上的通道。前往地上的过程是一趟有来无回的旅途，前往地上之后，存在于Metaverse之中的身体将会彻底消失。但是回到地上之后的人类仍旧拥有可以操纵Metaverse留在地球上的残余机构的权限。
